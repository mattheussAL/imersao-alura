## How Staring

<div>
  
  ```
  git clone https://github.com/mattheussAL/quiz-star-wars.git
  ```

  ## Start project in your machine
  ``` 
    npm install
    npm run dev
  ```
  OR
  ``` 
    yarn 
    yarn dev
  ```
